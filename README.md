# SpringCloudDemo，一个综合Spring Cloud各组件的综合Demo
## 项目结构说明
* base
-- api返回实体基类
-- 包含mybatis generator插件
-- yml共用配置
-- pom公用模块
* base-config
-- 微服务配置中心
-- 包含配置中心安全配置
* base-eureka
-- 服务注册中心
-- 包含agent代码替换功能测试
* base-gateway
-- 服务网关
-- 集成shiro,redis
-- 提供接口调用统计样板
* java-agent-demo
* service-api
* service-api2
* service-job
* service-mq
* service-multiple-cache
* web-test
