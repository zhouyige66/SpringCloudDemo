# SpringCloudDemo，一个综合Spring Cloud各组件的综合Demo
## 项目结构说明
* base
1. api返回实体基类
2. 包含mybatis generator插件
3. yml共用配置
4. pom公用模块
* base-config
1. 微服务配置中心
2. 包含配置中心安全配置
* base-eureka
1. 服务注册中心
2. 包含agent代码替换功能测试
* base-gateway
1. 服务网关
2. 集成shiro,redis
3. 提供接口调用统计样板
* java-agent-demo
* service-api
* service-api2
* service-job
* service-mq
* service-multiple-cache
* web-test
